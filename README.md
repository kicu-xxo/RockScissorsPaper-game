## 🎠구현 설명🎠
✌ 사용자가 '가위', '바위', '보' 버튼 중 하나를 누르면 컴퓨터는 랜덤한 결과를 리턴합니다.

✊ 선택된 결과에 따라 이름, 이미지, 승무패 값이 나타납니다.

🖐 결과를 확인하기 쉽도록 테두리 색이 변합니다
